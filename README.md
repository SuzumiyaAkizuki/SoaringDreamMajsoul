# 飞梦雀赛成绩统计器

这是一个基于 Python 和 NiceGUI 开发的轻量级 Web 工具，专为飞梦雀赛赛事管理设计。它可以自动读取赛事后台导出的牌谱记录，通过白名单过滤无关对局，并自动计算参赛选手的最终成绩与排名。

## 核心功能

- 同时支持「四人麻将」和「三人麻将」的成绩独立计算与展示。
- 未报名选手参与的对局正常计算，但在输出最终名单时删除未报名选手
- 自动按成绩降序排列。四麻前 16 名、三麻前 9 名自动高亮为浅绿色（晋级区）；对局场次不足及未晋级选手高亮为浅红色。
- 支持将包含所有对局明细的完整成绩单下载为 `CSV` 格式（采用 `UTF-8-SIG` 编码，原生兼容 Excel 且无中文乱码）。

## 环境依赖与运行

本项目无需复杂的环境配置。请确保已安装 Python 3.8+，并在终端执行以下命令安装依赖：

Bash

```
pip install pandas nicegui
```

**运行程序：**

在项目目录下运行以下命令：

Bash

```
python mahjong_stats.py
```

终端提示启动成功后，在浏览器中访问 [http://127.0.0.1:1146](https://www.google.com/search?q=http://127.0.0.1:1146) 即可使用。

## 输入文件格式要求

**所有上传的文件必须为 `.csv` 格式**。工具内置了编码自适应模块，兼容 `utf-8`、`gbk` 等多种常见编码。

### 1. 白名单 (Namelist)

- **文件格式**：`.csv`

- **内容要求**：

  - 程序会读取文件的第一列作为合法的选手 ID。
  - 无须表头（第一行即视为有效数据）。
  - 程序在匹配时会自动忽略大小写并清除首尾空格。
  - 暂未区分报名三麻或四麻。只要报了名，就算是同时报名了三麻和四麻。

- **示例**：

  ```
  选手A
  Player_B
  雀魂萌新
  ...
  ```

### 2. 牌谱记录 (四麻 / 三麻)

- **文件格式**：`.csv`
- **获取方式**：直接从「雀魂赛事管理后台 -> 赛事牌谱」导出。
- **内容要求**：请勿随意删改官方导出的列顺序。程序强依赖特定的列索引提取数据：
  - **四麻**：玩家昵称在第 4、7、10、13 列；得点在第 6、9、12、15 列。
  - **三麻**：玩家昵称在第 4、7、10 列；得点在第 6、9、12 列。

## 成绩计算规则

本工具内置了飞梦雀赛的算分逻辑，具体如下：

1. **有效场次判定**：选手必须完成**至少 5 场**有效对局（无未报名选手参与的对局），否则总分判定为 `对局场次不足`，并在最终排名中置底。
2. **最高均分**：在成绩列表中，分别计算取前 5 场、前 6 场……最高取至前 20 场的**平均分**。
3. **最终得分**：取上述所有组合均分中的**最大值**作为该选手的最终总分。

## 注意事项

- **前端展示**：为了保证页面流畅度和美观，网页端的表格仅展示「排名」、「最终成绩」和「用户名」。
- **明细数据**：点击页面右侧的「下载成绩表 (.CSV)」按钮，下载的本地文件中将包含该选手每一场的具体得分记录。
